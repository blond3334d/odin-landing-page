# odin-landing-page

This is a project where I get to apply both the HTML and CSS concepts I have learned by making an entire web page.

The page is for a fictional theater/cinema business called CineMarahuyo. It is the landing page where informations about the theater are presented in a somehow pleasing way using CSS.

<b>Areas where I had difficulty on:</b>
- Box model
- Margin and Padding setting
- Making the content/page adjustable based on different screen sizes

<b>Things that I've learned throughout:</b>
- The importance of specificity because I encountered a lot of problems with some of my declarations not being applied.
- Also learned about the browser's default setting that can override rules/declarations.
- The importance of arrangement or the order of writing the code as well as the importance of knowing what elements/contents to put in one container.

Overall, this project has been a great experience to learn and apply my acquired skills on the TOP Foundations course so far. Absolutely stressful yet fun --- exactly how I like things to be.
